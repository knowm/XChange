Bitcoinium Web Service API specification
================================

Public API:

Documentation
------
https://bitcoinium.com/service/home

Ticker History
------
https://bitcoinium.com/service/tickerhistory?pair=BTC_USD&timewindow=1h

Ticker
------
https://bitcoinium.com/service/tickerupdate?pair=MTGOX_BTC_USD

Orders
------
https://bitcoinium.com/service/orderbook?exchange=mtgox&pair=MTGOX_BTC_USD&pricewindow=10p
