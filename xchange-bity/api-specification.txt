Bity Exchange API specification
================================

Documentation
-------------
No documentation is available.
This code has been created by inspecting api calls on the bity website.

The auth method used is OAuth2 with grant_type = password.

Credentials :
API-Key = Bity username
API-Secret = Bity password

You need an extra parameter for client Id :
exSpec.setExchangeSpecificParametersItem("clientId", "XXXXXXXXX");
