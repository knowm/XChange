BitcoinToYou Exchange API specification
================================

Documentation
-------------
- [BitcoinToYou API](https://www.bitcointoyou.com/DocumentacaoAPI.aspx)

API Support
-----------
- support@bitcointoyou.zendesk.com

Limitations (according to documentation)
--------------
- Minimum interval for each call for info API: 10 secs
- Maximum API calls per day: 100 (or aprox. 0.07 / min in 24h) - Contact them if you need an exception.
